# Masonry Work Module (Zednické práce)

Specializace: Zdivo, příčky, obklady

## Typické práce:
- Zdivo nosné (cihly, bloky)
- Příčky (pěnobeton, sádrokarton)
- Obklady stěn
- Dlažby
- Komíny
- Příčky akustické

## 📊 TABULKY PRODUKTIVITY (ZDIVO)

### Zdění (m²/h na 1 zedníka):

Načíst z B3/productivity_rates.json:

```yaml
Cihla plná pálená:
  rychlost_m2_h: 0.6
  rychlost_m3_h: 0.15  # při tl. 250mm
  brigáda: 3  # 2 zedníci + 1 pomocník
  m2_den: 14.4
  tloustka: 250mm
  poznámka: "Tradiční, pomalé, kvalitní"

Cihla děrovaná:
  rychlost_m2_h: 0.73
  rychlost_m3_h: 0.18
  brigáda: 3
  m2_den: 17.5
  tloustka: 250mm
  poznámka: "Lehčí než plná"

Bloky betonové:
  rychlost_m2_h: 0.83
  rychlost_m3_h: 0.25
  brigáda: 3
  m2_den: 20
  tloustka: 300mm
  poznámka: "Větší prvky = rychlejší"

Pěnobeton (Ytong, Porotherm):
  rychlost_m2_h: 1.0
  rychlost_m3_h: 0.4
  brigáda: 2  # Lehké, 2 stačí
  m2_den: 16
  tloustka: 400mm
  poznámka: "Nejrychlejší, snadná manipulace"
```

## 🔧 TECHNOLOGIE ZDĚNÍ

### Standardní postup:

1. **Příprava podkladu**
   - Očištění, vyrovnání
   - Založení první vrstvy (důležité!)

2. **Zdění**
   - Nanášení malty
   - Kladení prvků
   - Kontrola svislosti a vodorovnosti

3. **Spárování**
   - Vyplnění spár
   - Úprava povrchu

4. **Osazení překlady**
   - Nad otvory (okna, dveře)

## 📐 FAKTORY OVLIVŇUJÍCÍ PRODUKTIVITU

### 1. Výška zdiva:

```yaml
do_3m: 
  faktor: 1.0  # normální rychlost

3-6m:
  faktor: 0.85  # potřeba lešení
  extra_náklady: "Lešení 150 Kč/m²/den"

nad_6m:
  faktor: 0.7  # složitá manipulace
  extra_náklady: "Lešení + výtah"
```

### 2. Otvory (okna, dveře):

```yaml
žádné_otvory:
  faktor: 1.0

málo_otvorů_<10%:
  faktor: 0.95  # malý vliv

hodně_otvorů_>20%:
  faktor: 0.85  # časté přerušování
  extra: "Překlady nad každým otvorem"
```

### 3. Vazba zdiva:

```yaml
běžná_vazba:
  faktor: 1.0
  příklady: ["běžná", "křížová"]

složitá_vazba:
  faktor: 0.8
  příklady: ["architektonická", "dekorativní"]
```

## 🚧 SPECIÁLNÍ PŘÍPADY

### Pěnobeton (Ytong, Porotherm):

```yaml
Výhody:
  - Nejrychlejší (1.0 m²/h)
  - Lehký (2 zedníci stačí)
  - Přesné rozměry
  - Tenká malta

Speciální:
  - Lepící malta místo běžné
  - Potřeba speciální pily
  - Kontrola rovinnosti důležitá
```

### Příčky (nenosné):

```yaml
Typy:
  - Pěnobeton: 1.2 m²/h (velmi rychle)
  - Sádrokarton: 2.0 m²/h (nejrychlejší)
  - Cihla děrovaná: 0.8 m²/h

Poznámka:
  - Nenosné → lze rychleji
  - Menší kontrola svislosti
  - Tenčí (100-150mm)
```

## 🎯 DECISION TREE

```
START: Zednická práce
  ↓
1. Určit typ materiálu
  ├─ Cihla plná → 0.6 m²/h, tradicná
  ├─ Cihla děrovaná → 0.73 m²/h
  ├─ Bloky betonové → 0.83 m²/h
  ├─ Pěnobeton → 1.0 m²/h, nejrychlejší
  └─ Speciální → Hledat podobné
  ↓
2. Určit faktory
  ├─ Výška: <3m / 3-6m / >6m
  ├─ Otvory: žádné / málo / hodně
  └─ Vazba: běžná / složitá
  ↓
3. Vypočítat adjusted rychlost
  rychlost = base × faktor_výška × faktor_otvory × faktor_vazba
  ↓
4. Vypočítat čas
  čas_h = plocha_m2 / (brigáda × rychlost_adjusted)
  čas_dny = čas_h / (brigáda × 8h)
  ↓
5. Vypočítat materiály
  ├─ Prvky (cihly/bloky): závisí na rozměru
  ├─ Malta: ~2% objemu zdiva
  └─ Překlady: podle otvorů
  ↓
6. Pomocné
  ├─ Lešení: pokud >3m
  ├─ Výtah: pokud >6m
  └─ Speciální nástroje
  ↓
END: JSON output
```

## KRITICKÉ FAKTORY

### FLAG as AMBER:
- Neznámý material (hledat podobný)
- Výška >6m (složitá logistika)
- Hodně otvorů >20%
- Speciální vazba

### FLAG as RED:
- Material není v databázi
- Složitá architektura
- Extrémní výška >12m
- Nestandardní požadavky
